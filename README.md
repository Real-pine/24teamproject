## ⚙기능설명

스크립트 별 설명

|스크립트|해당 스크립트 상세 설명|
|---|:---|
|Program.cs|객체초기화, 필수 데이터 설정 및 관리, 기본값 설정, UI표시기능, 포션관련기능관리, 세이브로드 관련 기능 관리, 사운드 설정 기능|
|SoundManager.cs|사운드관련 생성자 함수관리|
|Pub.cs|마을UI기능, 숙박기능, 음주기능, 동전던지기기능, 경마기능, 블랙잭기능|
|Battle.cs|데미지계산로직, 플레이어기본공격기능, 플레이어스킬공격, 몬스터공격기능|
|DungeonPlay|던전 진행에 관련한 맵db관리, 몬스터db관리, 던전진행에 따른 메서드들을 관리|
|Stage.cs|맵을 위한 생성자 함수 관리, 맵에서 플레이어 이동기능|
|Monster.cs|몬스터생성자 함수 관리|
|CharacterCustom.cs|캐릭터생성자 함수 관리, 인벤토리관리기능, 장착관리기능|
|Item.cs|아이템생성자 함수 관리|
|Quest.cs|퀘스트UI, 퀘스트보상, 퀘스트진행상황관리|
|SaveLoad.cs|세이브,로드 기능 및 세이브목록관리|
